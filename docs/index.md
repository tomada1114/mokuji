# mokuji

The most readable way to browse Markdown in your terminal.

**mokuji** (目次, "table of contents") is a terminal Markdown reader built
with [Textual](https://textual.textualize.io/). Run it in any repository
and read its documentation with Vim-style keys, a polished dark theme,
and an always-visible key guide.

## Installation

=== "uv"

    ```bash
    uv tool install mokuji
    ```

=== "pipx"

    ```bash
    pipx install mokuji
    ```

=== "pip"

    ```bash
    pip install mokuji
    ```

## Quick Example

```bash
cd your-project
mokuji                # browse the repository
mokuji docs/usage.md  # open one file directly
```

Press `?` inside the app for the full key reference.
