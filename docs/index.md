# mokuji

A short description of what this library does.

## Installation

=== "pip"

    ```bash
    pip install mokuji
    ```

=== "uv"

    ```bash
    uv add mokuji
    ```

## Quick Example

```python
from mokuji import add

result = add(1, 2)
print(result)  # 3
```
